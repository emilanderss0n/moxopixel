Transform the default Escape From Tarkov menu with this custom mod! Inspired by the Arena mode, this overhaul brings a new level of immersion by adding your player character to the first menu screen.

Key features include

  • Menu design with your player character featured on the main screen
  • Options to toggle on or off character level display, character model rotation, and more
  • Increase game longetivity thanks to a more inspiring menu layout than the default EFT menu screen
